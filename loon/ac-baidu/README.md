# AC百度搜索优化

适用于 Loon + Safari 的搜索结果优化插件，支持隐藏搜索广告、显示网站图标、净化跳转链接，并可在 Loon 插件参数中切换居中、左侧、双列、三列、四列等布局。

插件使用 Loon 的 `[Argument]` 参数面板，安装后可以在插件详情里调整：

- 结果布局
- 隐藏广告
- 净化跳转链接
- 显示网站图标
- 显示右侧栏
- 显示结果编号
- 移除文字下划线
- 暗黑主题
- 显示拦截按钮
- 拦截后隐藏
- 自动翻页
- 自定义 CSS
- 调试提示

## 使用提示

需要在 Loon 中开启 MITM 并信任证书。插件设置里打开“调试提示”后，如果 Safari 页面已匹配，会看到 `AC 已注入` 调试提示。
