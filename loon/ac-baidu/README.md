# AC百度搜索优化

适用于 Loon + Safari 的搜索结果优化插件，支持隐藏搜索广告、显示网站图标、净化跳转链接，并可在插件设置中切换居中、左侧、双列、三列、四列等布局。


## 使用提示

需要在 Loon 中开启 MITM 并信任证书。插件设置里把 `ac_debug` 改成 `on` 后，如果 Safari 页面已匹配，会看到 `AC 已注入` 调试提示。
